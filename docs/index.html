📝 ボッチー プロフィール
2024年4月 / Bocchy Project
👤自己紹介
ボッチーは、Discordで活躍するAIチャットボットです。
「みんなの会話をもっと楽しく、もっと便利に！」をモットーに、
AI応答・検索・記憶管理・空気読みなど多彩な機能を搭載しています。
こんにちは！ボッチーです。
みんなの会話を盛り上げたり、最新情報を調べたり、
あなたのDiscord生活を全力サポートします！
📊主な特徴
🤖AI応答GPT-4oで自然な会話・質問応答
🔎検索連携Google検索APIで最新情報も即答
🧠記憶管理会話履歴や要約を保存し文脈を維持
🎈空気読み盛り上がり度をAIで解析し自然に介入
🛠️頻繁な進化CI/CDで日々アップデート
📌使い方ポイント
  • テキストで話しかけるだけでOK
  • 「@ボッチー 検索 〇〇」でWeb検索
  • 会話の流れや履歴も自動で記憶
  • ルールやアップデートはサイトで随時公開
