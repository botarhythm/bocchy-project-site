ボッチーの最初のバージョンを公開しました。AIによる自然な会話と最新情報検索、記憶管理、空気読み機能を搭載しています。OpenAI GPT-4oを活用した高度な会話能力、Google検索連携による情報取得、Supabaseを使った記憶システムにより、ユーザーとの対話をより自然で有益なものにしています。
