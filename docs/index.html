開発サマリー

ボッチーの最初のバージョンを公開しました。AIによる自然な会話と最新情報検索、記憶管理、 空気読み機能を搭載しています。OpenAI GPT-4oを活用した高度な会話能力、Google検索連携による情報取得、 Supabaseを使った記憶システムにより、ユーザーとの対話をより自然で有益なものにしています。

自然対話処理フロー

ユーザー入力
Discordチャットからの入力を受け取り、処理の開始点となります。
意図理解
GPT-4oがユーザーの質問や指示の意図を分析し、返答に必要な情報や処理を判断します。
情報検索判断
外部情報が必要か、既存の知識で対応可能かを判断します。最新情報や具体的事実の確認には検索機能を活用します。
Google検索連携
Custom Search APIを使って関連情報を検索し、結果を取得します。検索結果を基に回答を生成するための情報を補完します。
記憶参照
過去の会話履歴をSupabaseから取得し、文脈理解や一貫性のある会話を実現します。ユーザーごとの対話履歴を管理します。
空気読み判定
会話の盛り上がり度や適切な対応レベルを分析し、状況に合わせたトーンと内容で応答します。
応答生成・送信
収集した情報を基に、最適な応答を生成しDiscordに送信します。同時に会話履歴を記憶として保存します。
